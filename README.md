# EHR System Project

## QUICK START

**New to the project? Start here:**

1. Read **INSTALLATION_GUIDE.md** for complete setup instructions
2. Follow the step-by-step guide to install WAMP/XAMPP
3. Set up the database and configure the project
4. Start using the EHR system

## Directory Structure

 ehr-system/
│
├── css/ # Stylesheets
│
├── js/ # JavaScript files
│
├── images/ # Static images (logos, icons, etc.)
│
├── includes/ # Reusable PHP includes
│
├── modules/ # Main application modules
│ ├── auth/ # Authentication (login, logout, register)
│ ├── patients/ # Patient management
│ ├── appointments/ # Appointment scheduling
│ ├── doctors/ # Doctor management
│ └── dashboard/ # Dashboard pages
│
├── config/ # Configuration files
│
├── database/ # Database files (SQL schema)
│
└── uploads/ # User uploaded files
├── documents/ # Medical documents, reports
└── profile_pics/ # Profile pictures



## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP
- Database: MySQL
